🐱 MeowMart Services (Backend API)

MeowMart Services adalah backend service untuk sistem manajemen petshop berbasis web.
Backend ini dibangun menggunakan Laravel dan menyediakan REST API untuk mendukung fitur POS, manajemen produk, booking, dan transfer stok antar cabang.

🚀 Tech Stack

- Laravel
- PHP
- MySQL
- Middleware Authentication
- Database Migration & Seeder

🧩 Fitur Utama

- 🔐 Authentication & Middleware
- 🛒 API Point of Sale (POS)
- 📦 Manajemen Produk
- 🔄 Transfer Stok Antar Cabang
- 📊 Download Laporan Penjualan
- 🗂 CRUD Data Master
- 🧪 Database Migration & Seeder
- 🧠 Role & Responsibility

Dalam project ini saya bertanggung jawab untuk:

- Mendesain struktur database
- Membuat migration & seeder
- Mengembangkan REST API controller
- Implementasi middleware
- Mengembangkan fitur laporan dan download report
- Integrasi dengan frontend (Vue 3)
